TP - API GraphQL avec Spring Boot

Contexte

Vous devez développer une API GraphQL pour une application e-commerce permettant de gérer un catalogue de produits avec catégories et avis clients.

Modèle de données

Category (1) ──────< (N) Product (1) ──────< (N) Review
Category Product Review
id id id
name name author
description description comment
price rating
stock createdAt
category_id product_id

Travail demandé

Partie 1 - Setup

  1. Créer un projet Spring Boot avec les dépendances : spring-boot-starter-graphql, spring-boot-starter-web, spring-boot-starter-data-jpa, h2, lombok
  2. Configurer H2 et GraphiQL dans application.yml

Partie 2 - Implémentation

  1. Créer les 3 entités JPA avec leurs relations
  2. Créer les repositories
  3. Créer les services avec opérations CRUD
  4. Créer les controllers GraphQL
  5. Définir le schéma GraphQL (schema.graphqls)

Partie 3 - Tests
